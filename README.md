## ✨ Features

- 🌌 **3 sectors / levels**, each with 3 escalating enemy waves
- 👹 **Boss fight** at the end of every level (Sentinel Drone → Void Harvester → Galaxy Devourer)
- 🎵 **Procedural chiptune music & sound effects** — fully synthesized live with the Web Audio API, zero audio files
- 🪐 **Animated background** — parallax starfield, drifting nebula clouds, slow-moving planets, occasional shooting stars
- 👾 Pixel-art enemies and ship, hand-drawn on canvas grids
- ❤️ Lives, score, level HUD, pause menu, mute toggle
- 💗 Power-up drops (extra life)
- 📱 Touch controls for mobile, full keyboard support for desktop
- ⏳ Animated loading screen on first launch

## 🕹️ Controls

| Action | Key |
|---|---|
| Move | `←` `→` or `A` `D` |
| Fire | `Space` |
| Pause | `P` |
| Mute | `M` |

Touch controls appear automatically on mobile/touch devices.
